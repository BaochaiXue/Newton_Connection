# Spec: fast_foundation_stereo

## Goal

Evaluate whether Fast-FoundationStereo is worth adopting for the cases this
project actually uses.

## Non-Goals

- Generic benchmark tourism
- Adopting it without a case-relevant comparison

## Inputs

- `docs/bridge/tasks/fast_foundation_stereo.md`
- available data/modalities
- current RealSense depth path

## Outputs

- one bounded conclusion:
  adopt / study further / not worth it now
- at least one successful case or a clear failure mode

## Constraints

- focus on cases that matter to this project
- compare against current depth-path value, not hypothetical future value

## Done When

- one explicit project-level conclusion is recorded
- at least one success or failure case is documented
