# Task Contracts

This directory stores milestone-level contracts for non-trivial work.

Use a contract when an agent should define what “done” means for one milestone
before implementation begins.

Start from:

- [_contract_template.md](./_contract_template.md)

Recommended usage:

- one contract per milestone, not one giant contract for the whole project
- link the contract from the matching task status or plan when it becomes active
