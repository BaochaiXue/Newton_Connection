# Results Metadata Rules

This subtree mirrors authoritative meaning for local result bundles whose heavy
artifacts may remain outside git.

## Rule

Authoritative claim meaning must not live only in ignored local binaries.

## Expected Surfaces

- `INDEX.md`
- `LATEST.md`
- `DEPRECATED.md` when superseded runs matter
- machine-readable run metadata under `runs/`

Use repo-relative paths whenever possible.
