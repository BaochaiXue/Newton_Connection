# Evaluations

This directory stores reusable evaluation rubrics and validation rules for
Codex-driven work.

Evaluator surfaces are selectively load-bearing:

- required for high-risk result-bearing or meeting-facing tasks where automatic
  PASS is not enough
- optional for lightweight tasks that have simple, objective validation

The repo expresses this through existing task contracts, skeptical review
bundles, verdict surfaces, and closeout lint rather than a second evaluation
framework.
