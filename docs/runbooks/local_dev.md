# Local Development Runbook

## Purpose

Explain how to work in this repo without relying on memory.

## Standard Start

1. Read `AGENTS.md`
2. Read `docs/README.md`
3. Read `TODO.md`
4. Open the relevant task page under `docs/bridge/tasks/`

## For Non-Trivial Work

Create or update:

- `tasks/spec/<task>.md`
- `plans/active/<task>.md`
- `tasks/implement/<task>.md`
- `tasks/status/<task>.md`

## Preferred Command Style

Use canonical wrappers under `scripts/` when available.
